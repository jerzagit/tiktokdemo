# Projek TikTok Demo

Struktur Projek:

```
DATABASE_README.md
api/
    register.php
config/
    database.php
database_schema.sql
html2.php
index.php
readme.md
sample_data.sql
sample.json
```

## Penerangan Folder & Fail

- **api/**: Mengandungi skrip API, contoh: `register.php` untuk pendaftaran pengguna.
- **config/**: Fail konfigurasi, contoh: `database.php` untuk sambungan ke pangkalan data.
- **database_schema.sql**: Skrip SQL untuk membina struktur jadual dalam pangkalan data.
- **sample_data.sql**: Data contoh untuk dimasukkan ke dalam pangkalan data.
- **sample.json**: Data contoh dalam format JSON.
- **index.php**: Halaman utama aplikasi.
- **html2.php**: Contoh halaman HTML tambahan.
- **DATABASE_README.md**: Dokumentasi berkaitan pangkalan data.
- **readme.md**: Fail ini (penerangan projek).

## Cara Guna Projek

1. Import `database_schema.sql` ke dalam MySQL untuk cipta jadual.
2. Import `sample_data.sql` untuk data contoh.
3. Edit `config/database.php` untuk tetapan sambungan database.
4. Jalankan aplikasi melalui `index.php`.

## Teknologi Digunakan
- PHP
- MySQL
- HTML, CSS, Javascript
